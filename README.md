# Monolith Architecture sample
> NestJs + TypeOrm + GQL (code-first)


## Setting

```bash
nest new project-name

# typeorm
yarn add typeorm mysql2 @nestjs/typeorm
# nest@8 ~ nestjs/gql@9 ~ apollo@3
yarn add @nestjs/graphql graphql apollo-server-express
```
